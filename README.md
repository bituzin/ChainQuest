ChainQuest

Praktyczna platforma do nauki smart kontraktów.

Krótko:
- Przeglądaj zadania, pisz i testuj rozwiązania, zdobywaj punkty i odznaki.
- Zawiera smart kontrakty (Solidity), frontend React (Vite + Tailwind) i skrypty Hardhat.

Szybkie uruchomienie:
- Zainstaluj zależności (`npm install`) w odpowiednich pakietach.
- Uruchom lokalny node/Hardhat i frontend (instrukcje w katalogach `packages`).

Wkład:
- Wysyłaj PR na gałąź `working`.
- Nie dodawaj sekretów do repo — używaj `.env` i `.gitignore`.

To krótki opis; szczegóły konfiguracji i użycia znajdziesz w katalogach projektu.
